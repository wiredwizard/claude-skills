# LinkedIn Batch Apply - 2 Jobs Nationwide

Automate applying to 2 remote LinkedIn jobs nationwide with your location set to Harrisburg, PA.

## Features

- **2 Remote Jobs Per Run** - Applies to 2 qualifying remote positions per execution
- **Nationwide Search** - Searches remote opportunities across the entire country
- **Location Profile** - Sets your location to Harrisburg, PA
- **Auto-Fill** - Automatically fills all form fields with your resume data
- **Smart Filtering** - Rejects hybrid/on-site roles and Resolv Global positions
- **Adaptive Answers** - Intelligently answers custom questions based on job requirements

## Installation

1. Download the skill package
2. Extract to your Claude skills folder
3. Install via Claude Settings > Capabilities > Skills Manager
